# 安装

1. 下载flutter源代码

   仓库地址： https://github.com/flutter/flutter

2. 配置环境路径

   ```bash
   export PATH=`pwd`/flutter/bin:$PATH
   ```

   

3. 运行flutter doctor

   运行以下命令查看是否需要安装其他依赖

   ```bash
   flutter doctor
   ```
