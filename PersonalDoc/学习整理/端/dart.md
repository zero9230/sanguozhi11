# 语法

## 变量

脚本类型语言，可以不预先定义变量类型，自动推断，使用var声明，也可以用具体类型声明，如

```dart
var str='hello';
String str='hello';
int a=123;

//分段长字符串
String str='''123
456
789
''';


//取值
var name='张三';
print("$name")
  
  int b=10;
	b??=23;//b为空，则赋值；不为空则不执行
```
