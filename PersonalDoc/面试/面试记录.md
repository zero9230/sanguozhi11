# 百度如流

死锁

```text
四个条件
1. 互斥条件（Mutual exclusion）：资源不能被共享，只能由一个进程使用。
2. 请求与保持条件（Hold and wait）：已经得到资源的进程可以再次申请新的资源。
3. 非剥夺条件（No pre-emption）：已经分配的资源不能从相应的进程中被强制地剥夺。
4. 循环等待条件（Circular wait）：系统中若干进程组成环路，该环路中每个进程都在等待相邻进程正占用的资源。
```
